Integration

API & SDK Documentation

Integrate OmniaGuard into your existing security stack with our RESTful API and native SDKs.

REST API

# Get device security status
GET /api/v1/devices/{id}/status
# Trigger threat scan
POST /api/v1/scans
# Get compliance score
GET /api/v1/compliance/{framework}
# Deploy ShadowGrid decoy
POST /api/v1/shadowgrid/decoys

Full API documentation available to partners. Rate limits: 1000 req/min (Enterprise), 10000 req/min (MSSP).
